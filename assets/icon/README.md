# Інструкція для підготовки іконок

## Необхідні файли

Для генерації всіх необхідних іконок потрібно покласти сюди два файли:

### 1. `icon.png` - Основна іконка
- **Розмір:** 1024x1024 px (мінімум, можна більше)
- **Формат:** PNG
- **Зміст:** Повна іконка з синім круглим фоном та білим символом (ланка ланцюга зі стрілкою)
- **Фон:** Синій круг (не прозорий)

### 2. `icon_foreground.png` - Для адаптивних іконок Android
- **Розмір:** 1024x1024 px (мінімум, можна більше)
- **Формат:** PNG з прозорим фоном
- **Зміст:** Тільки білий символ (ланка ланцюга зі стрілкою) БЕЗ синього фону
- **Фон:** Прозорий (alpha channel)

## Після додавання файлів

Після того, як покладете обидва файли в цю папку, запустіть:

```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

Це автоматично згенерує всі необхідні розміри іконок для:
- Android (всі розміри + адаптивні іконки)
- iOS (всі розміри)

## Додатково для Google Play Store

Для завантаження в Google Play Console також потрібна іконка:
- **Розмір:** 512x512 px
- **Формат:** PNG (32-бітний)
- **Зміст:** Та сама іконка з синім фоном

Можете створити її з `icon.png`, зменшивши до 512x512 px.
